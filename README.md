# SCOPE
This repository holds data and codes to reproduce numerical and graphical results from the manuscript "Systematic modeling of phenotypic drug response profiles in patient-derived organoids". The codes are written in MATLAB. 

Spreadsheet data containing the results of NN analysis of organoid images are provided in separate folders, arranged by which figures in the manuscript they belong to.

All codes are provided together in a single folder. The code with the name "FigX" reproduces Figure X in the manuscript.
